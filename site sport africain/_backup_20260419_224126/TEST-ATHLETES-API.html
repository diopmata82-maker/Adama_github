🧪 Test API Athlètes - Sport Africain

Système dynamique pour afficher les informations des sportifs africains

1. Afficher un athlète

Fonction: displayAthlete(name, containerId)

2. Afficher plusieurs athlètes

Fonction: displayAthletesList(names, containerId)

3. Récupérer les données (console)

Fonction: getAthlete(name)

Ouvrez la console (F12) pour voir les résultats

📖 Guide d'utilisation

1. Inclure le script

<script src="athletes-api.js"></script>

2. Afficher un athlète

// HTML
<div id="athlete-info"></div>

// JavaScript
displayAthlete('Sadio Mané', 'athlete-info');

3. Afficher une liste

// HTML
<div id="athletes-list"></div>

// JavaScript
const names = ['Sadio Mané', 'Joel Embiid', 'Eliud Kipchoge'];
displayAthletesList(names, 'athletes-list');

4. Récupérer les données

const athlete = await getAthlete('Sadio Mané');
console.log(athlete.name);
console.log(athlete.description);
console.log(athlete.image);
