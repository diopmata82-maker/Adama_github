# Guide des Images - Sport Africain

## Section "Nos Sports"

Pour que chaque sport soit visuellement identifiable, veuillez ajouter les images suivantes :

### Images requises :

1. **Football** : `podcast-football.jpg`
   - Image d'un joueur de football africain ou d'un match de football
   - Dimensions recommandées : 800x800px minimum
   - Format : JPG

2. **Basketball** : `podcast-basketball.jpg`
   - ✅ Images déjà fournies (3 variantes disponibles)
   - Utiliser l'une des images fournies

3. **Athlétisme** : `podcast-athletisme.jpg`
   - Image d'un athlète africain en course ou saut
   - Dimensions recommandées : 800x800px minimum
   - Format : JPG

4. **Lutte** : `podcast-lutte.jpg`
   - Image de lutteurs africains en action
   - Dimensions recommandées : 800x800px minimum
   - Format : JPG

### Autres sections :

#### Section Actualités (6 images)

#### Section Interviews (3 images)

### Notes importantes :
- Toutes les images doivent être optimisées pour le web
- Format recommandé : JPG (meilleure compression)
- Qualité : 80-85% pour un bon équilibre poids/qualité
- Assurez-vous que les images sont libres de droits ou que vous avez les autorisations nécessaires
