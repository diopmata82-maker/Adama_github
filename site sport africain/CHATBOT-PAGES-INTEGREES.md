## 🐛 Dépannage

### Le chatbot n'apparaît pas ?

1. **Vérifiez la console** : F12 → Console
2. **Vérifiez les fichiers** : chatbot.js et chatbot.css doivent être dans le même dossier
3. **Rechargez la page** : Ctrl+F5 (force refresh)
4. **Vérifiez le code** : Les 2 lignes doivent être avant `</body>`

### Le bouton est là mais ne s'ouvre pas ?

1. Vérifiez la console pour les erreurs JavaScript
2. Assurez-vous que chatbot.js est bien chargé
3. Essayez sur une autre page

### Les liens ne fonctionnent pas ?

1. Vérifiez que les pages cibles existent
2. Vérifiez les chemins des liens dans chatbot.js
3. Testez les liens manuellement
